# 焙草集 AI 销售助手 — 更新说明

**日期：** 2026年5月23日  
**版本：** MVP v0.2  

---

## 一、新增特性

### 1. 对话自动审核系统

**是什么：** AI 每天自动检查所有对话，找出问题并给出改进建议。

**怎么用：** 打开后台 → 对话 → 审核报告 → 点击「运行审核」，AI 会分析最近的对话并生成报告，包括：
- 质量评分（1-10分）
- 发现的问题（按高危/中危/低危分级）
- 改进建议（话术、流程、知识库）

**历史对比：** 左侧时间线保存每次审核结果，可以对比不同时间的改进效果。

**价值：** 不用逐条翻对话，AI 告诉你哪有问题、怎么改。

---

### 2. 对话详情 + 店员标记

**是什么：** 每条对话都可以点开看完整记录，店员可以标记和备注。

**怎么用：**
- 后台 → 对话 → 点击任意对话的「详情」按钮
- 右侧滑出完整对话历史（顾客和 AI 的每一句话）
- 底部可以写备注、打标签（如"问题对话""已处理""好评"）
- 点击「标记」按钮把对话标为重点

**筛选：** 可以按「已标记」筛选，只看需要关注的对话。

**价值：** 发现问题对话 → 标记 → 备注 → 后续跟踪改进。

---

### 3. 渠道追踪

**是什么：** 追踪顾客从哪个渠道来的（不同门店、不同二维码）。

**怎么用：** 小程序分享二维码时带上渠道参数（如 `?channel=wechat-dongmen`），后台自动统计每个渠道的对话数。

**查看：** 后台 → 对话 → 每条对话显示来源渠道。后台 → 报表可看到渠道统计。

**价值：** 知道哪个门店/渠道的顾客最多，哪个渠道需要加强运营。

---

### 4. 10种体质套餐全覆盖

**是什么：** 全面支持中医10种体质（气虚、阳虚、阴虚、血虚、痰湿、湿热、血瘀、气郁、特禀、平和），每种都有推荐产品套餐。

**怎么用：** 后台 → 体质套餐 → 切换体质标签 → 为每种体质配置推荐产品（面包+零食+茶饮搭配）。

**套餐数量：** 每种体质 6-7 件产品，店员可自行增减。

**价值：** 覆盖所有体质类型，不再只有5种。产品搭配更丰富，顾客购买更多。

---

### 5. 体质对话智能识别（两阶段定位）

**是什么：** AI 用更聪明的方式判断顾客体质，不再像问卷一样逐个问题问。

**流程：**
1. 先让顾客自由描述身体感受（1轮）
2. AI 根据描述缩小到 2-3 个候选体质
3. 再问 1-2 个鉴别问题确认（如"你更怕冷还是身体更沉？"）
4. 锁定体质，推荐产品

**对比旧版：** 旧版逐个问 5 个固定问题（"怕不怕冷？""上不上火？"...），现在最多 2-3 轮就搞定。

**价值：** 对话更快、更自然，顾客体验更好。

---

### 6. 店员修正体质 + 体质转化报表

**是什么：**
- 如果 AI 判断的体质不准确，店员可以手动修正
- 后台新增体质转化报表，看每种体质的顾客买了多少

**怎么用：**
- 修正体质：后台 → 对话 → 点开详情 → 顶部绿色条显示体质 → 下拉选择修正
- 查看报表：后台报表中显示各体质对话数、订单数、转化率

**价值：** AI 判断 + 人工复核，越来越准。数据驱动运营。

## 二、优化改进

### 1. 对话速度大幅提升

- 切换 AI 模型：从 MiniMax M2.7 换成 DeepSeek V4
- **响应速度提升 60-70%**（原来 10-30 秒，现在 1-5 秒）
- 不再出现超时和卡顿

### 2. 对话质量优化

- 去掉了"怎么称呼你呀"这个冗余环节（微信授权直接获取顾客昵称）
- 体质提问不再重复（之前会问两次"怕不怕冷"）
- 对话流程：打招呼 → 安全筛查 → 体质判断 → 困扰确认 → 推荐，不再卡在某一步

### 3. 后台界面改进

- 审核报告：左侧时间线 + 右侧详情，点击切换
- 对话详情：改成右侧抽屉，不遮挡列表
- 体质套餐：10 种体质全部展示，标签自动换行
- 分类筛选：修复 `[object Object]` 显示问题

### 4. 测试与质量保障

- 6 位模拟顾客完整对话测试（小明、小花、小张、小李、小白、小黄）
- 78 个测试用例覆盖所有 API 和 UI
- QA 测试报告自动生成

## 四、使用建议

### 店员日常使用流程

1. **看审核报告**：每天打开后台 → 对话 → 审核报告，看 AI 发现了什么问题
2. **标记问题对话**：在对话列表中标记有问题的对话，写备注
3. **修正体质**：如果发现 AI 判断不准，手动修正顾客体质
4. **调整套餐**：根据实际情况调整每种体质的推荐产品

### 老板关注指标

1. **AI 转化率**：报表 → AI 转化率，看 AI 推荐是否带来订单
2. **体质分布**：报表 → 顾客体质分布，了解顾客群体特征
3. **审核评分**：对话 → 审核报告 → 质量评分，追踪 AI 服务质量变化
4. **渠道效果**：哪个渠道带来的对话最多
