## 根因分析

### FAIL-001: LLM API 认证失败（影响 T13-T16, T25）
- **原因:** `backend/.env` 中 `llm_api_key` 无效或过期
- **修复:** 更新 MiniMax API key

### FAIL-002: Excel 导入 500（T12）
- **原因:** `biscuit.xlsx` 中文编码乱码，导入逻辑无法解析
- **修复:** 从有赞重新导出（确保 UTF-8 编码），或手动修复 Excel

### FAIL-003: 顾客详情 JSON 解析失败（T34）
- **原因:** SQLAlchemy 返回的 Customer 对象含有乱码中文，FastAPI JSON 序列化失败
- **修复:** 确保数据库中的中文数据编码正确

### W-001: 分类筛选不匹配（T3）
- **原因:** 商品分类名乱码（如 "鐐?缂?" 而非 "tea"），`?category=tea` 无法匹配
- **影响:** 前端分类筛选功能部分失效

### W-002: SKU 查询无结果（T5）
- **原因:** Excel 中的自定义 SKU（B001, T001）不在导入数据中，实际 SKU 为有赞格式（P260316...）
- **影响:** 需使用实际 SKU ID 查询
