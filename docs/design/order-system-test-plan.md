## 每个顾客的流程

```
1. greeting → empty message
2. screening → "都没有"
3. constitution describe → 体质特征描述
4. constitution differential → 鉴别回答（如有）
5. scene → 困扰描述
6. recommend → "推荐产品"
7. CREATE ORDER（关联对话+推荐+体质快照）
```
